# AI Staff Panel

Dedicated staff frontend cloned from `ai-company`.

## Login Source
- Staff are created from `ai-company` in the `Staff` tab.
- Use staff email + login password set during staff create/edit.

## Setup
1. `cd ai-staff`
2. `npm install`
3. Copy `.env.example` to `.env` if needed

## Run
- `npm run dev`

Frontend: `http://localhost:5175`  
Backend API: `http://localhost:5000/api`
